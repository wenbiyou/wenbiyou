## 🚀 开源项目

### [ai-ui-vue](https://github.com/wenbiyou/ai-ui-vue)
**Vue3 AI 应用专用组件库**  
为 AI 场景量身打造，包含流式对话、智能表单、Markdown 渲染、Token 计数等组件，让 AI 应用开发更快更省心。

### [Remi](https://github.com/wenbiyou/Remi)
**本地化 RAG 知识管理系统**  
个人本地知识库，基于 RAG 技术智能检索碎片化信息，保护隐私的同时提升知识利用率。

**微前端实践**  
qiankun + Vite + Vue3 完整方案，包含样式隔离、主从通信、状态管理。

**局域网代码同步工具**  
保持格式，拒绝乱码，局域网内快速同步代码片段。

### [frontend-rag-knowledge-base](https://github.com/wenbiyou/frontend-rag-knowledge-base)
**前端知识库 RAG**  
用 RAG 技术构建的前端知识库，方便快速检索开发知识。

### [interview-practice](https://github.com/wenbiyou/interview-practice)
